# Data Science Portfolio

6. Predicting Fuel Efficiency

7. Using KNN to Cluster Loan Applicants

8. Predicting Baseball Attendance
- Description: Regression/time-series analysis predicting game attendance; includes data merging (schedules, weather), EDA, and model evaluation.
- Showcases: data-prep notes; model comparison (baseline vs. advanced); and sample predictions versus actual attendance.

9. Visualizing Netflix Media Popularity
- Description: Exploratory analysis and visualizations of Netflix titles' popularity over time and by category/region.
- Showcases: polished static visualizations; interactive chart examples; and key insights summarizing trends.

10. Visualizing Video Game Sales
- Description: Sales analysis across platforms and regions with EDA and relative ranking of best-selling titles.
- Showcases: comparative visualizations; concise business-oriented takeaways; and reproducible EDA steps.
